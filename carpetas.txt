3.- css : contiene los scrips css que serviran para dar estilo personalizado en caso de ser util
4.- img : contiene todas las imagenes que necesitemos agregar a nueestras vistas
5.- archivos : contiene todas las imagenes de los productos agregados
6.- js: contiene todos lo scripts js para cada midulo para ejecutar las funciones de crud
7.- librerias: contendra lubrerias para la creacion de pdf, jquery, alertify, bootstrap y select 2
8.- procesos : contiene archivos intermedios entre las clases y las vistas, estos mandaran informacion entre una y otra cosa
9.- contendra todas las vistas de cada uno de los modulos
